# RPGVendor
ReactJS app deployable as .exe file with Electron. Simple RPG game vendor with nice UI.
It's using simple json file database.

Used technologies:
- React + Redux
- Electron
- SCSS

Main functionalities:
- statistics
- cash
- buying items
- selling items
- different item types
- eating food from eq
- disassembling items
